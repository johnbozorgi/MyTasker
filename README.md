ğŸš€ Installation & Setup Guide

To use this project on your browser as a developer, follow these steps:

Download the Project:
Clone this repository or download the ZIP file and extract it.

git clone [https://github.com/johnbozorgi/MyTasker.git](https://github.com/johnbozorgi/MyTasker.git)


Open Extension Management in Chrome:
Type chrome://extensions in your browser's address bar and hit Enter.

Enable Developer Mode:
Toggle the Developer mode switch in the top right corner.

Load the Extension:
Click the Load unpacked button (top left).
Select the folder containing the project files (manifest.json, popup.html, etc.).

Test:
The MyTasker icon will now appear in your Chrome toolbar. Click it to open the dashboard.
